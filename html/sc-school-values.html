School Values
General Information
  • To turn on School Values: Super Admin > District Options > Use School Values
  • To edit the content: Admin > School Values
  • To customize the labels: School Constants > School Values
  • To make it School Values a mapping category: Admin > All Mapping Categories
    • Like the "catch-all" Standards and Assessment, there should be only ONE School Values Catgory in ONE Atlas system.
    • Until you make School Values a mapping category and place it on the template, School Values always shows as a tab on this template.
Edit School Values
  1. To edit Activity drop-down list: Admin > School Values > Activities
  2. To edit School Values drop-down list: Admin > School Values > Core School Value Types and School Values
Rename Labels
  1. To Rename "Actvity": School Constants > School Values > sOpportunityLabel + sOpportunitiesLabel
  2. To Rename "Description": School Constants > School Values > sSchoolValueAssessmentDescriptionLabel
  3. To Rename "School Values": School Constants > School Values > sSchoolValue + sSchoolValues
School Values Category
3 Options in All Mapping Categories
3 Options on Template
Search Options
  1. TRUE: School Constants > School Values > bMultiLevelSchoolValuesSearchByType
  2. FALSE: School Constants > School Values > bMultiLevelSchoolValuesSearchByType
