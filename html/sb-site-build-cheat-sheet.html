Site Build Cheat Sheet
Push Standards
  • When adding State standards in Amalgamation, make sure to add the whole set based on State Standards Push
  • Choose one and only one set of standards to be the "structure", usually state standards, CCSS, or IB...
  • Enter added standards names to the “Notes” box on the Summary page.
  • Check the SET box if SET is turned on, and check the Script box if script is used.
